PROJE ALT YAPISI
Proje Node.js + Express + EJS ile yapılmaktadır (React kullanılmaz).
Frontend EJS view dosyaları ve CSS ile yapılır.
Çoklu dil desteği TR / PL / UA i18n kullanılacaktır.

HEDEF
1) Sistem menüsünü global olarak gizlenebilir hale getir. Menü default olarak gizli olsun, kullanıcı hamburger butonuna basınca açılsın. Tüm sayfalarda bu davranış geçerli olsun.
2) Depo stok listesi sayfasında PL ve UA çevirilerini i18n’e ekle.

1) MENÜ DAVRANIŞI (GLOBAL)
- Layout dosyasında (ör. views/layout.ejs) menü paneli default olarak collapsed (gizli) olacak.
- Sayfanın sol üst köşesinde ☰ (hamburger) butonu olacak.
- Butona basıldığında menü slide-out panel gibi açılacak.
- Menü açıkken sağ kısım gri arka plan overlay ile kapanacak, kullanıcı boşluğa tıklayınca kapanacak.
- Bu davranış tüm sistem sayfaları için geçerli olacak (dashboard, ürünler, stoklar, raporlar).
- Menü durumunu (açık/kapalı) localStorage’da tut, kullanıcı sayfa değiştirince aynı kalsın.

2) DEPO STOK LİSTESİ (views/stock_by_warehouse.ejs)
- Barkod bazlı listeleme kalsın (her barkod ayrı satır).
- Filtre paneli: Depo seçimi + raf seçimi + ürün adı araması.
- Tablo kolonları: Ürün, Özellikler, Miktar, Barkod, Raf.
- Satıra tıklayınca modal açılır, barkod detayları görünür.
- Modalda “Yazdır” ve “Export” butonları olacak.
- TR çevirilere ek olarak PL ve UA çevirileri de i18n’de tanımlansın.
