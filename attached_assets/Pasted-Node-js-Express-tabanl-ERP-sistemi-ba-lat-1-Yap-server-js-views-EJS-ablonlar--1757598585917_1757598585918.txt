Node.js + Express tabanlı ERP sistemi başlat. 

1) Yapı:
   /server.js
   /views (EJS şablonları)
   /public (css, js, uploads)
   /data (örnek json veriler)
   /locales (tr.json, pl.json, ua.json)

2) Paketler:
   express, ejs, body-parser, i18n, uuid, chart.js (cdn ile frontend), nodemon

3) server.js:
   - Express başlat (port 5000)
   - view engine: ejs
   - public klasörünü static sun
   - i18n yapılandır (locales: ["tr","pl","ua"], defaultLocale:"tr", queryParameter:"lang")
   - Routes:
     GET "/" → dashboard sayfası
     GET "/products" → ürün kartları (ileride)
     Diğer menü başlıkları için boş placeholder sayfalar (örn: /stock, /sales)

7) views/layout.ejs:
   - Ortak layout (bootstrap navbar + sidebar menü)
   - Menü TR/PL/UA butonları (query param ile lang değiştirme)

8) views/dashboard.ejs:
   - Başlık: <%= __("dashboard.title") %>
   - Üstte 4 kutucuk: Toplam Ürün, Aktif Ürün, Düşük Stok, Toplam Değer
   - Altında 2 grafik (Chart.js):
     a) Son 6 ay satış grafiği (dummy data)
     b) Kategoriye göre ürün dağılımı (dummy data pie chart)

9) /data/products.json:
   - 5 örnek ürün (id, name, buyPrice, sellPrice, stock, status, category)

10) Sonuç:
   - "/" sayfası Dashboard olarak açılacak.

   - Dashboard’ta özet kutular + 2 grafik olacak.
