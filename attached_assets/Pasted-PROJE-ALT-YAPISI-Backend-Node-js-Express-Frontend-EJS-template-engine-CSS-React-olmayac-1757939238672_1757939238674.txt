PROJE ALT YAPISI
- Backend: Node.js + Express
- Frontend: EJS template engine + CSS (React olmayacak)
- Çoklu dil desteği TR / PL / UA (i18n)
- Veritabanı: müşteri kartları, hareketler (satış, ödeme, iade), bakiyeler

HEDEF
ERP seviyesinde müşteri yönetim modülü oluştur:
- Müşteri kartı
- Cari hesap hareketleri
- Ödeme alma
- Müşteri istatistikleri (hacim, sadakat, aktif/pasif)

1) MÜŞTERİ KARTI
Form alanları:
- [ ] Müşteri kodunu otomatik oluştur (checkbox)
   • Checkbox işaretli → sistem 6 haneli sayı üretir (100000’den başlar, +1 artar, 00 ile başlamaz)
   • Checkbox boş → kullanıcı manuel girer
- Müşteri Adı (zorunlu)
- Telefon
- Adres
- Para Birimi (USD, PLN, UAH, TRY)
- Açılış Bakiyesi
- [Kaydet]

2) MÜŞTERİ LİSTESİ
Tablo sütunları:
- Müşteri Kodu
- Müşteri Adı
- Telefon
- Bakiye (Borç kırmızı, Alacak yeşil)
- İşlemler: Düzenle | Cari Hesap Gör | Ödeme Al | Sil

3) CARİ HESAP
- Müşteri detay sayfasında borç/alacak/bakiye özeti
- Hareket tablosu:
   • Tarih
   • İşlem Türü (Satış, Ödeme, İade)
   • Açıklama
   • Borç
   • Alacak
   • Kalan Bakiye
- Filtre: tarih aralığı + işlem türü
- Export CSV/PDF, Yazdır

4) ÖDEME AL
- Müşteri listesinde veya hesap sayfasında “Ödeme Al” butonu
- Modal: Tutar, Para Birimi, Açıklama, Tarih
- Kaydet → hareket cari hesaba eklenir, bakiye güncellenir

5) MÜŞTERİ İSTATİSTİKLERİ
- KPI Kartları:
   • Toplam müşteri
   • Toplam borç
   • Toplam alacak
   • En çok alış yapan müşteri
   • En çok ödeme yapan müşteri
- Grafikler:
   • Çubuk: En çok işlem yapan ilk 10 müşteri
   • Pasta: Sadakat dağılımı (Altın / Gümüş / Bronz)
   • Çizgi: Son 12 ay müşteri hareketleri
- Tablo:
   • Müşteri kodu
   • Adı
   • Toplam Satış
   • Toplam Ödeme
   • Bakiye
   • İşlem Sayısı
   • Sadakat seviyesi
- Sadakat Seviyesi:
   • Altın: yüksek hacim + düzenli işlem
   • Gümüş: orta
   • Bronz: düşük veya pasif

6) TASARIM
- Modern dashboard görünümü
- Gri arka plan, beyaz kartlar
- Responsive
- Renkli rozetler (borç/alacak, sadakat)
